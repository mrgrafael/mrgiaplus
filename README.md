# React CRM

Este é um projeto React para CRM hospedado em VPS com Ubuntu e Nginx.

## Como rodar

1. Clone o repositório:
   ```sh
   git clone https://github.com/seuusuario/react-crm.git
   cd react-crm
   ```

2. Instale as dependências:
   ```sh
   npm install
   ```

3. Para rodar em desenvolvimento:
   ```sh
   npm run dev
   ```

4. Para build e rodar no servidor:
   ```sh
   npm run build
   npm start
   ```
